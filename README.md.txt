# 📊 Mobile Sales Dashboard

An interactive Power BI dashboard built using Microsoft Excel data to analyze mobile phone sales across different cities, brands, payment methods, customer ratings, and time periods.

---

## 🚀 Project Overview

This dashboard helps visualize and analyze mobile sales performance using interactive charts and filters. It provides business insights into sales trends, customer behavior, and product performance.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

---

## 📈 Dashboard Features

- KPI Cards
  - Total Sales
  - Total Quantity Sold
  - Total Transactions
  - Average Sales

- Interactive Month Filter

- Sales by Mobile Model

- Sales by City (Map Visualization)

- Customer Ratings Distribution

- Payment Method Analysis

- Day-wise Sales Trends

## 💡 Key Insights

- Total Sales: **₹68M**
- Total Quantity Sold: **2K Units**
- Total Transactions: **331**
- Average Sale Value: **₹41K**
- Apple and Samsung contribute significantly to overall sales.
- UPI, Debit Card, Credit Card, and Cash are the primary payment methods.
- Delhi, Mumbai, Hyderabad, and Lucknow recorded strong sales performance.
